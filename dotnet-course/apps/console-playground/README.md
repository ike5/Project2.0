# Sample App: `console-playground`

A small console app demonstrating C# fundamentals, used across Modules 01–04. Each
demo maps to a module so you can run the concept you're learning.

## Run
```bash
cd dotnet-course/apps/console-playground
dotnet run            # shows a menu; pick 1-4 or 'a' for all
dotnet run -- 3       # run demo #3 (LINQ) directly
```

## Demos
| # | Demo | Module |
|---|------|--------|
| 1 | Types & nullability | 01 |
| 2 | OOP: classes, records, interfaces | 02 |
| 3 | Generics, LINQ, delegates, pattern matching | 03 |
| 4 | async/await | 04 |

The source under `Demos/` is intentionally short and commented — read it alongside
each module's `README.md`.
